# User Story Format

```
As a <role>
I want <capability>
So that <benefit>
```

## Acceptance Criteria format (Given/When/Then)

```
Given <context>
When <action>
Then <outcome>
```

## Fields per story

- Story ID
- Epic
- Story (As a / I want / So that)
- Acceptance Criteria
- Priority (High / Medium / Low)
- Story Points
- Complexity
